flink 在 OSX 系统上以 local 方式运行

Published Wed, Mar 14, 2018 by Ruhm

安装

$ brew install apache-flink

$ flink -v
Version: 1.3.2, Commit ID: 0399bee

$ brew info apache-flink
apache-flink: stable 1.3.2, HEAD
Scalable batch and stream data processing
https://flink.apache.org/
/usr/local/Cellar/apache-flink/1.3.2 (108 files, 156.6MB) *
  Built from source on 2018-03-13 at 19:31:16

启动

$ cd /usr/local/Cellar/apache-flink/1.3.2
$ ./libexec/bin/start-local.sh
Starting jobmanager daemon on host alastairruhm.local.
