pyenv 通过国内镜像安装 python

sudo curl -L https://raw.githubusercontent.com/yyuu/pyenv-installer/master/bin/pyenv-installer | bash

export v=3.5.1
wget http://mirrors.sohu.com/python/$v/Python-$v.tar.xz -P ~/.pyenv/cache/;pyenv install $v

TIPS:

  1. 这里使用的是搜狐的镜像源：http://mirrors.sohu.com/python/
  2. 下载python压缩包放到~/.pyenv/cache文件夹下面
  3. 执行 pyenv install 版本号 安装对应的python版本
